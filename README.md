# Personal Portfolio Website

## Overview
This is a responsive Personal Portfolio Website developed as part of Task-1 of the ApexPlanet Full Stack Web Development Internship Program.

The website showcases personal information, skills, projects, and contact details using HTML, CSS, and JavaScript.

## Features
- Responsive Design
- Navigation Bar
- Home Section
- About Section
- Skills Section
- Projects Section
- Contact Form
- Form Validation using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

```
portfolio/
│
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```

## How to Run

1. Download the project.
2. Extract the folder.
3. Open `index.html` in a browser.

OR

1. Place the project folder inside XAMPP's `htdocs` folder.
2. Start Apache from XAMPP.
3. Open:

```
http://localhost/portfolio
```
